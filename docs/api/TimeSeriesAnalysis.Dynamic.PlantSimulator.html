    Class PlantSimulator

    Simulates larger "plant-models" that is built up connected sub-models, that each implement ISimulatableModel

    To set up a simulation, first connect models, and then add external input signals. This class handles information about which model is connected to which, and handles callig sub-models in the correct order with the correct input signals.

    By default, the model attempts to start in steady-state, intalization handled by ProcessSimulatorInitalizer (this requires no user interaction)

    The building blocks of plant models are PIDModel, DefaultProcessModel and Select
